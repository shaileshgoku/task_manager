# 📝 Task Manager (Streamlit App)

A simple and interactive Task Manager web app built using Python and Streamlit.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✅ Mark tasks as complete
* ❌ Delete tasks
* 💾 Persistent storage using JSON

---

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON (for data storage)

---

## ▶️ Run Locally

```bash
streamlit run streamlit_app.py
```
